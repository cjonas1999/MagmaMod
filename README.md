# MagmaMod
Small hollow knight mod to show activation range of dog cycle bug.

# How to use
Includes ability to toggle the outline with a hotkey (default F9).

This keybind can be updated in the file at `%USERPROFILE%\AppData\LocalLow\Team Cherry\Hollow Knight\MagmaMod` using the names listed at the [bottom of this link](https://docs.unity3d.com/6000.3/Documentation/ScriptReference/KeyCode.html).
